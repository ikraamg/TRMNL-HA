# TRMNL Home Assistant Add-ons

Custom add-ons for Home Assistant optimized for TRMNL e-ink displays.

## TRMNL HA

Send dashboard screens to your TRMNL e-ink display.

![UI Screenshot](trmnl-ha/example/ui.png)

[Read the README](https://github.com/ikraamg/home-assistant-addons/blob/main/trmnl-ha/README.md) for full documentation and configuration options.
